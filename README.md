# Food Delivery Website

## Project Overview
This project is a food delivery website that allows users to easily browse and order food from a variety of restaurants in their area. The application aims to provide a seamless and user-friendly experience for both customers and restaurant owners.

## Features
- User Registration and Login
- Browse Restaurants
- Search for Food Items
- Add Items to Cart
- Secure Checkout Process
- Order Tracking
- Restaurant Dashboard for Managing Menu and Orders

## Technologies Used
- Frontend: HTML, CSS, JavaScript, React
- Backend: Node.js, Express.js
- Database: MongoDB
- Deployment: Heroku / AWS

## Installation Instructions
1. Clone the repository: `git clone https://github.com/DurgeshSChauhan007/Tomato_FoodDeliveryWebsite.git`
2. Navigate to the project directory: `cd Tomato_FoodDeliveryWebsite`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
